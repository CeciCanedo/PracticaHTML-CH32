TikTok

Tiktok es una plataforma que se basa en la creacion de videos para compartirlos. Tiene 740 millones de usuarios.

La empresa que lanzo esta aplicacion en 2016 en China fue ByteDance, es un software de uso gratuito y no es posible acceder a su codigo fuente.

Lista de tecnologias

  • Java
  • Python
  • C
  • Swift

Instagram

Instagram es una aplicacion que permite tomar fotos, editarlas y compartirlas. Actualmente es propiedad de meta y tiene 120 millones de usuarios.

Esta aplicacion fue lanzada originalmente unicamente para los dispositivos aple el 6 de octubre de 2010.

Lista de tecnologias

  • El codigo fuente es privado pero se sabe que el backend depende de Django (framework escrito en Python), Celery, Redis y Cassandra.
  • La base de datos usada es PostgreSQL y las notificaciones push a cargo de NodeJS.

Facebook

Facebook es en realidad un conglomerado de redes sociales y tecnologia. Permite a sus usuarios crear albumes y postear en su biografia, ademas de compartir lo que otros publican.

Cuenta con 2.910 millones de usuarios mensuales fue lanzada como "The Facebook" en 2004 por Zuckerberg, Severin, Moskovit y Hughes.

Lista de tecnologias

  • Linux como sistema operativo
  • Apache Cassandra para almacenar datos de manera estructurada
  • Apache Hive para la busqueda
  • Apache Thrift para acelerar el desarrollo e implementar seervicios de backend eficientes y escalables
  • Esta escrito en los lenguajes de PHP, Java, C++, Erlang, .NET y PEARL

Whatsapp

Whatsapp es la aplicacion de mensajeria mas utilizada en el mundo. Su principal caracteristica es la mensajeria instantanea y funcionalidades extra como estados que duran 24 horas paraa que tus contactos telefonicos registrados puedan verlos.

Esta aplicacion es parte de Meta y fue lanzada en noviembre de 2009.

Lista de tecnologias

  • Emplea lenguaje Erlang y XML
  • Con FunXMPP, una variante de XMPP, los mensajes tienen menor peso
  • Entre los servidores se encuentran FreeBSD para el sistema operativo con tecnologia PHP y Yaws como servidor web
  • Los contenidos se comparten mediante Whisper y son cifrados por BASE 64 y RADIX 64 thumbnails

Telegram

La segunda aplicacion de mensajeria mas usada. Fue desarrollada por los hermanos rusos Nikolai y Pavel Durov.

Esta aplicacion permite unirte a grupos publicos de interes

Lista de tecnologias

  • Una parte del software es libre pero lo correspondiente al servidor es privado
  • Esta programada en API Java para Android
  • La interfaz grafica que tiene es framework Qt
